## What is MuteGuard?

MuteGuard is a lightweight Windows 11 audio control utility that puts your microphone and speaker mute toggles right at your fingertips — no more diving into system settings mid-call.

## ✨ Core Features

- **Dual Audio Control** — Independently mute mic and speaker with one click
- **Keyboard Shortcuts** — `Win + Alt + M` for mic, `Win + Alt + S` for speaker (fully rebindable)
- **Dual System Tray Icons** — Real-time mute state shown in the taskbar at all times
- **Floating Mini Window** — Always-on-top widget, draggable, scalable, position saved
- **Smart Device Detection** — Auto-scans all connected audio devices; easy dropdown switching
- **Dark Mode & Light Mode** — Matches Windows 11 native aesthetics
- **Run at Startup** — Launch with Windows automatically
- **Custom Icons** — Pick from any `.dll` or `.ico` file using native Windows picker
- **Persistent Settings** — All preferences saved to a local `.ini` file
- **Graceful Error Handling** — Retries 3× before surfacing tray notification

## 📥 How to Download & Install

### Step 1 — Go to the Releases Page

Click the button below or navigate to the **Releases** tab at the top of this repository page:

**[→ Go to Releases](https://github.com/xionraj/MuteGuard/releases/latest)**

---

### Step 2 — Choose Your Edition

On the latest release page, scroll down to the **Assets** section. You will see two files:

| File | Type | Best For |
|------|------|----------|
| `MuteGuard_vX.X_Setup.exe` | Installer | Most users — installs to your PC, adds Start Menu shortcut & uninstaller |
| `MuteGuard_vX.X_Portable.exe` | Portable | Advanced users — no installation, run from anywhere (USB, folder, etc.) |

> **Not sure which to pick?** Choose the **Installer** edition.

---

### Step 3 — Download the File

Click on the file name you want. Your browser will download it automatically.

> If your browser or Windows SmartScreen shows a warning, see **Step 5** below.

---

### Step 4a — Run the Installer *(if you chose the Setup edition)*

1. Open your **Downloads** folder and double-click `MuteGuard_vX.X_Setup.exe`.
2. If a **User Account Control (UAC)** prompt appears, click **Yes**.
3. Follow the on-screen setup wizard — click **Next** → **Install** → **Finish**.
4. MuteGuard will launch automatically and appear in your **System Tray** (bottom-right of taskbar).

---

### Step 4b — Run the Portable Version *(if you chose the Portable edition)*

1. Move `MuteGuard_vX.X_Portable.exe` to any folder you want (e.g. `C:\Tools\MuteGuard\`).
2. Double-click the `.exe` to launch it — no installation needed.
3. MuteGuard will appear in your **System Tray** (bottom-right of taskbar).
4. *(Optional)* To run at startup automatically, open MuteGuard Settings → enable **Run at Startup**.

---

### Step 5 — Windows SmartScreen Warning *(if it appears)*

Because MuteGuard is independently published, Windows may show a SmartScreen prompt on first run. This is normal for new apps without a paid code-signing certificate.

To proceed safely:

1. Click **"More info"** on the SmartScreen dialog.
2. Click **"Run anyway"**.
3. MuteGuard will launch normally.

> MuteGuard is open-source and contains no ads, trackers, or malware. You can review the full source on the [main repository](https://github.com/xionraj/MuteGuard).

---

### Step 6 — You're all set! 🎉

- **Right-click** either tray icon to access Settings, switch audio devices, or open Windows Sound Settings.
- **Left-click** the mic tray icon to instantly toggle mute.
- Use **`Win + Alt + M`** to mute/unmute your mic and **`Win + Alt + S`** for your speaker — from anywhere on your PC.
